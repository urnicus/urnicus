Maryland provides access to voter registration data and voter history at an affordable cost of $125. The state offers flexible delivery options via FTP or flash drive, with voter history available as either a combined file or separate download.

## Quick Summary

| Category | Details |
|----------|---------|
| **Statewide Cost** | $125 |
| **Voter List Access** | Online form request |
| **Voter History Access** | 10 elections (combined) or 5 elections (separate) |
| **Delivery Method** | FTP or flash drive |
| **Processing Time** | After form and payment processed |
| **Possible Requirement** | May need to be registered Maryland voter (verification needed) |

## Voter List

Maryland provides voter registration data through the State Board of Elections with an online request system.

### How to Access

1. Visit the [Maryland Elections Voter Registration Data page](https://elections.maryland.gov/voter_registration/data.html)
2. Complete the [online data request form](https://elections.maryland.gov/voter_registration/data_form.html)
3. Submit payment via the [Maryland SBE payment portal](https://www.velocitypayment.com/client/maryland/sbe/index.html) ($125)
4. Receive data via FTP or flash drive after processing

### What's Included

The voter registration file includes comprehensive information for registered voters in Maryland.

### Registration Requirement (Unconfirmed)

The PDF form includes a "registered in" field. It's unclear whether you must be a registered Maryland voter to obtain the list. The qualification language does not explicitly state this requirement.

**Recommendation:** Contact the State Board of Elections to confirm whether out-of-state requestors can purchase Maryland voter data.

## Voter History

Maryland provides voter history in two formats with different historical coverage depending on how you request it.

### Historical Coverage Options

| Format | Historical Coverage |
|--------|---------------------|
| **Combined with Voter List** | Last 10 elections |
| **Separate File** | Last 5 elections |

### How to Access

When completing the data request form, specify whether you want:
- Voter history combined with the voter registration list (10 elections), or
- Voter history as a separate file (5 elections)

## Cost

Maryland charges **$125** for voter registration data with voter history.

Payment is processed through the online payment portal: [Maryland SBE Payment Portal](https://www.velocitypayment.com/client/maryland/sbe/index.html)

## Delivery Options

Maryland offers flexible delivery methods:
- **FTP transfer**: Data uploaded to an FTP server you can access
- **Flash drive**: Physical media mailed to you

Specify your preferred delivery method on the request form.

## Restrictions and Permitted Uses

When requesting Maryland voter data, users must comply with state laws regarding the use of voter registration records. The request form will include specific terms of use and legal requirements.

## Technical Notes

### Voter History Format Choice

Maryland's two-format voter history system is unique:
- Choose **combined format** (10 elections) if you want voter registration and history in a single file
- Choose **separate format** (5 elections) if you prefer to join the files yourself or only need recent history

### Registration Question

Before submitting a request, consider calling the State Board of Elections to clarify:
- Whether you must be a registered Maryland voter
- Whether out-of-state requestors are eligible
- Any additional qualification requirements

### Testing Status

This system has not been independently tested. Contact the Maryland State Board of Elections for current information about:
