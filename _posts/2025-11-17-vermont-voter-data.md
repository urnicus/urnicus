Vermont provides free access to its statewide voter checklist through an email distribution list system. Once added to the list, you receive regular updates at no cost. However, Vermont requires registration as a data broker if you intend to share the voter information with others.

## Quick Summary

| Category | Details |

## Voter List

Vermont provides voter checklist data through multiple email distribution lists. Requestors can choose which list(s) they want to join.

### How to Access

1. Download and complete the [Vermont Statewide Checklist Request Form](https://outside.vermont.gov/dept/sos/Elections_Division/voters/faq/request_electronic_copy_vermont_statewide_checklist.pdf)
2. Submit the completed form to the Vermont Secretary of State
3. Select which distribution list(s) you want to join
4. You will be added to the selected list(s)
5. Receive voter data via email when distributed

### What's Included

The statewide checklist includes voter registration information for registered voters in Vermont.

## Voter History

The availability of voter history data is not explicitly stated in the request form.

### Recommendation

When submitting your request form, specifically ask whether voter history data is available and can be included with the voter checklist.

## Cost

Vermont's voter data appears to be **completely free**. There is no fee mentioned for joining the distribution lists.

## Data Broker Registration

> **Important Requirement:**
>
> If you intend to share voter information with others, Vermont requires registration as a data broker.

Review Vermont's data broker requirements before sharing any voter data obtained from the state.

## Restrictions and Permitted Uses

When accessing Vermont voter data, users must comply with state laws regarding the use of voter registration records, including data broker registration requirements if applicable.

## Technical Notes

### Distribution Method

Vermont operates multiple distribution lists. When you submit your request, you can select which specific lists you want to join based on your data needs.

### Delivery Format

Data is delivered via email to list subscribers when updates are available.

### Testing Status

This system has not been independently tested, but the process appears straightforward for qualified requestors.

Vermont's free voter data distribution list system provides no-cost access to statewide voter information. The data broker registration requirement for data sharing is an important consideration for users planning to redistribute the information.
