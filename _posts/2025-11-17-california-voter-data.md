California provides access to voter registration data through the VoteCal system. While not available for immediate online purchase, qualified applicants can obtain comprehensive voter data including voter history by submitting a mail-in request form.

## Quick Summary

| Category | Details |
|----------|---------|
| **Statewide Cost** | $100 |
| **Voter List Access** | Mail-in request form |
| **Voter History Access** | Included with voter list |
| **Delivery Method** | Contact SOS for delivery details |
| **Processing Time** | Varies after application review |
| **Eligibility** | Qualified Applicant requirements |

## Eligibility Requirements

California restricts access to voter data to "Qualified Applicants" as defined in the request form. Review the eligibility criteria carefully before submitting a request.

## Voter List

California provides voter registration data through the VoteCal system, managed by the Secretary of State's office.

### How to Access

1. Visit the [California VoteCal Project page](https://www.sos.ca.gov/elections/voter-registration/votecal-project)
2. Download the [Voter File Application PDF](https://elections.cdn.sos.ca.gov/pdfs/voter-file-application.pdf)
3. Review the Qualified Applicant requirements listed on the form
4. Complete the application form
5. Mail the completed form to the Secretary of State
6. Wait for application review and approval

### What's Included

The voter registration file includes comprehensive information for registered voters in California.

## Voter History

California includes voter history data with the voter registration file for qualified applicants.

### Historical Coverage

Voter history records are provided along with the voter registration data.

## Cost

California charges **$100** for statewide voter registration data with voter history.

## Restrictions and Permitted Uses

> **Important Legal Requirements:**
>
> Access is limited to "Qualified Applicants" as defined by California law. The application form details specific eligibility criteria that must be met.

When accessing California voter data, qualified applicants must comply with state laws regarding the use of voter registration records.

## Technical Notes

### Application Process

The request process requires:
1. Reviewing eligibility requirements
2. Completing the detailed application form
3. Mailing the application to the Secretary of State
4. Waiting for review and approval
5. Arranging payment and delivery

### Delivery Method

Contact the Secretary of State's office or review the application materials for information about how approved data requests are delivered.

### Testing Status

This system has not been independently tested. Contact the California Secretary of State's office for current information about:
