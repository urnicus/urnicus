South Dakota recently reduced its voter data fees from $2,500 to just $225 for a statewide electronic file. The state provides voter registration data and voter history through a simple mail-in request form at this newly affordable price.

## Quick Summary

| Category | Details |
|----------|---------|
| **Statewide Cost** | $225 (reduced from $2,500 in 2025) |
| **Voter List Access** | Mail-in request form |
| **Voter History Access** | Selectable checkbox on form |
| **Delivery Method** | Electronic file |
| **Processing Time** | After form and payment received |
| **Price Reduction** | Effective September 29, 2025 |

## Voter List

South Dakota provides voter registration data through the Secretary of State's office via a mail-in request form.

### How to Access

1. Download the [Voter File Request Form PDF](https://sdsos.gov/elections-voting/assets/VoterFileRequestForm.pdf)
2. Complete the request form
3. Check the voter history box if desired
4. Mail the form with payment ($225) to the address specified
5. Receive electronic file after processing

### What's Included

The voter registration file includes information for all registered voters in South Dakota.

## Voter History

South Dakota offers voter history as a selectable option on the request form.

### How to Access

When completing the Voter File Request Form, check the box for voter history to include it with your order.

### Historical Coverage

Contact the Secretary of State for details about the historical coverage period included in the voter history file.

## Cost

South Dakota's voter data pricing recently changed:

| Timeline | Cost |
|----------|------|
| **Before September 29, 2025** | $2,500 |
| **After September 29, 2025** | **$225** |

> **Significant Price Reduction:**
>
> As of September 29, 2025, the amended ARSD 5:04:06:07 lists the new fee for a statewide electronic file as $225.
>
> The regulation states: "The fees for an electronic file with voter registration information are as follows: (1) Statewide, two hundred twenty-five dollars".

This **90% price reduction** makes South Dakota voter data significantly more accessible than before.

## Restrictions and Permitted Uses

When requesting South Dakota voter data, users must comply with state laws regarding the use of voter registration records. The request form will include specific terms of use and legal requirements.

## Technical Notes

### Major Price Change

South Dakota implemented a dramatic fee reduction in late 2025:
- **Old price**: $2,500
- **New price**: $225
- **Savings**: $2,275 (90% reduction)

This change makes South Dakota one of the more affordable states for voter data access.

### Delivery Format

Data is provided as an **electronic file** after the request is processed.

### Request Form Checkbox

The voter file request form includes a checkbox option for voter history, making it easy to include historical data with your order.
