Washington State provides free access to its statewide voter registration database and voter history through an online request system. After a brief approval process, approved requestors receive immediate download access to comprehensive voter data at no cost.

## Voter List

Washington provides voter registration data through the Secretary of State's Voter Registration Database (VRDB) Extract system.

### How to Access

1. Visit the [Washington State Voter Registration Database page](https://www.sos.wa.gov/elections/data-research/reports-data-and-statistics/washington-state-voter-registration-database-vrdb)
2. Complete the [VRDB Extract Request Form](https://www.sos.wa.gov/washington-voter-registration-database-extract)
3. Submit your request online
4. Receive email confirmation and download link after approval
5. Download the data immediately

### What's Included

The VRDB extract includes comprehensive voter registration information for all registered voters in Washington State.

## Voter History

Washington includes voter history data with the voter registration database extract.

### Historical Coverage

Voter history records are included in the database extract, providing election participation data for registered voters.

## Cost

Washington's voter data is **completely free** to access and download.

## Processing Timeline

Based on testing:
- Form submitted online
- Email with download link received within minutes of approval
- Immediate download access

The approval process is typically very fast, with download links arriving shortly after submission.

## Restrictions and Permitted Uses

When accessing Washington voter data, users must comply with state laws regarding the use of voter registration records. Review the terms of use provided during the request process.

## Technical Notes

### Delivery Method

After approval, you receive an email with a direct download link to the voter database extract.

### File Format

Data is provided as a downloadable file accessible via the emailed link.

This system has been tested and verified to work correctly. The request form was submitted and a download link was received via email within minutes. The download process worked smoothly.
