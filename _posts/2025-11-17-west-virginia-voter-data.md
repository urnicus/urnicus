West Virginia provides access to voter registration data and voter history through a flexible pricing model. You can purchase the voter list for $500 and voter history for $500 separately, or choose a $1,000 annual subscription that includes both files with updates throughout the year.

| Category | Details |

| **Voter History Cost** | $500 (one-time) |

| **Voter History Access** | Available separately or with subscription |

| **Processing Time** | After form and payment received |

## Voter List

West Virginia provides voter registration data through the Secretary of State's office via a mail-in request form.

### How to Access

1. Visit the [West Virginia Elections page](https://sos.wv.gov/elections)
2. Download the [Voter Data Request Form PDF](https://sos.wv.gov/FormSearch/Elections/Informational/Voter%20Data%20Request.pdf)
3. Complete the request form
4. Select your purchase option:
   - Voter list only ($500)
   - Voter list + voter history ($1,000 one-time)
   - Annual subscription ($1,000 with updates)
5. Mail the form with payment to the address specified
6. Receive data via email or cloud storage

### What's Included

The voter registration file includes information for all registered voters in West Virginia.

## Subscription vs. One-Time Purchase

The subscription model offers significant value if you need current data:

**Subscription Advantages ($1,000/year):**
- Same price as buying both files once
- Receive updates throughout the year
- Maintain current voter data
- Get both voter history and registration updates

**One-Time Purchase ($500-$1,000):**
- Snapshot of data at purchase time
- No updates
- Lower cost if you only need one file
- Suitable for historical analysis

## Delivery Methods

West Virginia provides voter data via:
- **Email**: Data files sent via email
- **Cloud storage**: Access files via cloud storage link

This flexible delivery approach accommodates different file sizes and user preferences.

## Restrictions and Permitted Uses

> **Important Legal Requirement:**
>
> "No voter registration lists or data files containing voter names, addresses or other information derived from voter data files may be used for commercial or charitable solicitations or advertising, sold or reproduced for resale."

### Prohibited Uses
- Commercial solicitations
- Charitable solicitations
- Advertising
- Selling data
- Reproducing data for resale

When requesting West Virginia voter data, users must comply with these restrictions and other state laws regarding the use of voter registration records.
