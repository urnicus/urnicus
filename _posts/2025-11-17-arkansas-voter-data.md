Arkansas provides access to voter registration data and voter history at an exceptionally low cost of just $2.50 through a mail-in request form. The state offers flexible delivery options including CD, FTP, or Dropbox link.

## Quick Summary

| Category | Details |
|----------|---------|
| **Statewide Cost** | $2.50 |
| **Voter List Access** | Mail-in request form |
| **Voter History Access** | Available via same request form |
| **Delivery Method** | CD, FTP (requestor-provided), or Dropbox link |
| **Processing Time** | After form and payment received |

## Voter List

Arkansas provides voter registration data through the Secretary of State's office via a simple data request form.

### How to Access

1. Visit the [Arkansas Elections Research page](https://www.sos.arkansas.gov/elections/research/)
2. Download the [Data Request Form PDF](https://www.sos.arkansas.gov/uploads/elections/Data%20Request%20Form.pdf)
3. Complete the request form
4. Specify whether you want voter history data
5. Choose your delivery method (CD, FTP, or Dropbox)
6. Mail the form with payment ($2.50)
7. Receive data via your selected method

### What's Included

The voter registration file includes comprehensive information for all registered voters in Arkansas.

## Voter History

Arkansas provides voter history data through the same data request form as the voter registration list.

### How to Access

When completing the Data Request Form, indicate that you want voter history data included with your request.

### Historical Coverage

Contact the Secretary of State for details about the historical coverage period included in the voter history file.

## Delivery Methods

Arkansas offers three flexible delivery options:

**1. CD (Compact Disc)**
- Data delivered on physical media
- Mailed to your address
- No special technical setup required

**2. FTP (File Transfer Protocol)**
- Digital delivery via FTP server
- **You must provide your own FTP server**
- Faster than physical media
- Requires FTP hosting

**3. Dropbox Link**
- Data shared via Dropbox cloud storage
- Download at your convenience
- No special software required
- Faster than physical media

Choose the delivery method that best suits your technical capabilities and timeline.

## Restrictions and Permitted Uses

When requesting Arkansas voter data, users must comply with state laws regarding the use of voter registration records. The data request form will include specific terms of use and legal requirements.

## Technical Notes

### Exceptional Value

At $2.50, Arkansas offers:
- The lowest-cost voter data in the United States
- Complete statewide voter registration
- Voter history included
- Flexible delivery options

This pricing is 97% less expensive than the national average for voter data.

### FTP Delivery Requirement

If choosing FTP delivery, note that **you must provide your own FTP server**. The Arkansas Secretary of State will upload the data to your FTP server, but will not provide FTP hosting.

To use FTP delivery:
- Set up an FTP server or account
- Provide FTP credentials on the request form
- Ensure the server has sufficient storage space
- Test FTP access before submitting request

### Dropbox Option

The Dropbox link delivery method provides the best combination of:
- Speed (faster than CD delivery)
- Convenience (no FTP setup required)
- Accessibility (download when ready)
- No special software needed

### Mail-In Process

Arkansas uses a traditional mail-in request process rather than an online portal. This adds a few days to the overall timeline but ensures the low $2.50 price point remains accessible to everyone.
