Florida provides free access to its statewide voter registration list and voter history data through a monthly CD distribution program. While not available for immediate download, the state offers comprehensive voter data at no cost to qualified requestors.

## Quick Summary

| Category | Details |
|----------|---------|
| **Statewide Cost** | Free |
| **Voter List Access** | Monthly CD distribution |
| **Voter History Access** | Included with voter list |
| **Delivery Method** | CD mailed to requestor |
| **Processing Time** | Distributed on predetermined monthly schedule |

## Voter List

Florida releases updated voter registration data monthly on a predetermined schedule. The data is distributed on CD to approved requestors.

### How to Access

1. Visit the [Florida Division of Elections Voter Extract Request page](https://dos.fl.gov/elections/data-statistics/voter-registration-statistics/voter-extract-request/)
2. Send an email requesting to be added to the distribution list
3. You will receive confirmation when you're added to the list
4. A CD will be mailed on the next scheduled release date

### What's Included

The Full Voter Export includes comprehensive voter registration information for all registered voters in Florida.

## Voter History

Florida includes voter history data directly on the CD with the voter registration list, eliminating the need for a separate request.

### Historical Coverage

The voter history file is provided along with the voter registration data on the distributed CD.

## Cost

Florida's voter data is **completely free**. There is no charge for the CD or the data.

## Processing Timeline

Based on testing:
- Initial request submitted: October 15
- Confirmation of being added to list: October 27
- CD expected to be mailed: November 12 (next scheduled release)

Total timeline from request to receipt: Approximately 4 weeks

## Restrictions and Permitted Uses

When requesting Florida voter data, users must comply with state laws regarding the use of voter registration records. Review the Division of Elections requirements for permitted uses.

## Technical Notes

### Distribution Schedule

Florida operates on a predetermined monthly release schedule. Once added to the distribution list, you will automatically receive updated CDs on each release date.

### Delivery Format

Data is provided on physical CD media mailed to the address you provide in your request.

### Testing Status

This system has been tested. The confirmation process works efficiently, with email confirmation typically received within two weeks of the initial request.

---

Florida's free voter data system offers comprehensive statewide information at no cost, though the monthly CD distribution model requires planning ahead. The automatic inclusion on future distribution lists makes it convenient for ongoing access to updated data.
