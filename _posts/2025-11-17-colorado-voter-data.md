Colorado provides access to both voter registration lists and voter history data through a mail-in request system. Both files can be requested separately or together for a combined cost of $100.

## Quick Summary

| Category | Details |
|----------|---------|
| **Statewide Cost** | $50 (voter list) + $50 (voter history) = $100 total |
| **Voter List Access** | Mail-in request form |
| **Voter History Access** | Mail-in request form (separate file) |
| **Delivery Method** | Contact SOS for delivery details |
| **Processing Time** | Varies after request review |

## Voter List

Colorado provides voter registration data through the Secretary of State's office via a formal request process.

### How to Access

1. Download the [Data Request Form PDF](https://www.sos.state.co.us/pubs/elections/forms/dataRequests.pdf)
2. Complete the form for voter registration data
3. Include payment for $50
4. Mail the completed form with payment to the Secretary of State

### What's Included

The voter registration file includes information for registered voters in Colorado.

## Voter History

Colorado provides voter history as a separate data file that can be requested independently or along with the voter registration list.

### How to Access

1. Use the same [Data Request Form PDF](https://www.sos.state.co.us/pubs/elections/forms/dataRequests.pdf)
2. Check the option for voter history data
3. Include payment for $50 (or $100 if also requesting voter list)
4. Mail the completed form with payment to the Secretary of State

### Historical Coverage

Contact the Secretary of State for details about the historical coverage period included in the voter history file.

## Cost

Colorado's voter data pricing is structured as follows:

| Data Type | Cost |
|-----------|------|
| Voter Registration List | $50 |
| Voter History File | $50 |
| **Both Files Together** | **$100** |

Official fee schedule: [Colorado SOS Elections Fees](https://www.sos.state.co.us/pubs/info_center/fees/elections.html)

## Restrictions and Permitted Uses

When requesting Colorado voter data, users must comply with state laws regarding the use of voter registration records. The request form will include specific terms of use and legal requirements.

## Technical Notes

### Separate Files

Unlike many states, Colorado provides voter registration and voter history as separate files. You can:
- Request only the voter list ($50)
- Request only voter history ($50)
- Request both files together ($100)

### Delivery Method

Contact the Secretary of State's office or review the request form for information about how data is delivered after approval.

### Testing Status

This system has not been independently tested. Contact the Colorado Secretary of State's office for current information about:
- Processing timelines
- Delivery methods
- File formats
- Request requirements
