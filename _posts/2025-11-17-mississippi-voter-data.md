## Voter List

Mississippi distributes voter registration data through a weekly email program managed by the Secretary of State's office.

### How to Access

1. Review the [2025 Voter File Weekly Distribution Program details](https://www.sos.ms.gov/content/documents/about_us/2025%20Voter%20File%20Weekly%20Distribution%20Program.pdf)
2. Complete the [online signup form](https://app.smartsheet.com/b/form/5c8b666a679c4c148ab513c8981a4962)
3. Submit your information to be added to the distribution list
4. Receive weekly emails with download links to CSV files

### What's Included

The voter registration file includes comprehensive information for all registered voters in Mississippi.

## Voter History

Mississippi includes voter history data in the weekly email distribution along with the voter registration list.

### Historical Coverage

Voter history files are provided as part of the weekly data distribution, giving subscribers access to current participation records.

## Cost

Mississippi's voter data appears to be **completely free**. No fee is mentioned for joining the weekly distribution program.

## Update Frequency

The program provides **weekly updates**, ensuring subscribers have access to current voter registration data throughout the year.
