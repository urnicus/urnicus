North Carolina offers free access to both voter registration lists and voter history data through an online download portal. The state updates its files weekly and provides one of the most comprehensive voter history datasets available, covering the last 10 years of election participation.

| Category | Details |

| **Processing Time** | Instant |
| **Historical Coverage** | 10 years |

## Voter List

North Carolina provides voter registration data through the State Board of Elections website with weekly updates.

### How to Access

1. Visit the [NC State Board of Elections Voter Registration Data page](https://www.ncsbe.gov/results-data/voter-registration-data)
2. Access the [download portal](https://dl.ncsbe.gov/)
3. Download the current voter registration file
4. Files are updated weekly

### What's Included

The voter registration file includes comprehensive information for all registered voters in North Carolina.

## Voter History

North Carolina provides voter history as a separate download through the same portal system.

### How to Access

1. Visit the [NC Voter History Data page](https://www.ncsbe.gov/results-data/voter-history-data)
2. Download the voter history file
3. Files are updated weekly

### Historical Coverage

North Carolina's voter history files contain **10 years** of election participation records, providing extensive historical voting behavior data.

## Cost

Both the voter registration list and voter history data are **completely free** to access and download.

## Restrictions and Permitted Uses

When accessing North Carolina voter data, users should be aware of applicable state laws regarding the use of voter registration records. Check the State Board of Elections website for current terms of use and legal requirements.

## Technical Notes

### Update Frequency

Both voter registration and voter history files are updated weekly, ensuring access to current data.

### File Format

Files are provided as downloadable datasets through the state's download portal.

### Testing Status

This system has been tested and verified to work correctly. The process is straightforward with immediate access to current data.
