## Voter List

New York provides voter registration data through Freedom of Information Law (FOIL) requests submitted to the State Board of Elections.

### How to Access

1. Visit the [New York State Board of Elections FOIL Requests page](https://elections.ny.gov/freedom-information-law-foil-requests)
2. Review FOIL request procedures
3. Submit a FOIL request for voter registration data
4. Specify statewide or county/jurisdiction-level data
5. Indicate whether you want voter history included
6. Specify delivery preference (email or physical media)
7. Pay applicable fees (if any)
8. Receive voter data according to FOIL fulfillment procedures

### What's Included

The voter registration file should include information for registered voters in New York, subject to FOIL provisions and privacy protections.

## Voter History

New York provides voter history data through FOIL requests along with voter registration information.

### How to Access

When submitting your FOIL request, specify that you want voter history included with the voter registration data.

### Historical Coverage

Contact the State Board of Elections or include a question in your FOIL request about:
- What historical coverage period is provided
- File format for voter history
- How voter history matches to registration data
