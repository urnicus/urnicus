## Voter List

Nebraska provides voter registration data through the Secretary of State's online request system with an innovative update program.

### How to Access

1. Visit the [Nebraska Voter File Requests page](https://sos.nebraska.gov/elections/voter-file-requests)
2. Complete the online voter file affidavit
3. Submit the affidavit for review and approval
4. Pay the $500 fee
5. Receive the voter file
6. Request updated files anytime within one year

### What's Included

The voter file includes comprehensive voter registration information for all registered voters in Nebraska.

## Voter History

Nebraska includes voter history data with the voter registration file.

### Historical Coverage

Voter history records are provided along with the voter registration data. Contact the Secretary of State for details about the historical coverage period.

## Cost

Nebraska charges **$500** for the statewide voter file with voter history.

## Update Program

> **Excellent Update Policy:**
>
> Upon payment of the fee and approval of the voter file affidavit, you may request an updated file at any time for **one full year**. Nebraska runs a new file on the **first day of each month**.

This update policy provides exceptional value:
- One-year access to updated data
- Monthly file refreshes
- Request updates anytime during the year
- No additional fees for updates during the year

## Restrictions and Permitted Uses

When requesting Nebraska voter data, users must complete and sign an affidavit affirming compliance with state laws regarding the use of voter registration records.

The affidavit review process ensures requestors understand and agree to legal requirements for data use.

## Technical Notes

### Affidavit Requirement

Nebraska requires submission and approval of a voter file affidavit before providing access to voter data. This ensures:
- Requestors understand legal restrictions
- Proper use of voter data
- Compliance with state laws

### Monthly Updates

Nebraska maintains current voter data by running new statewide files on the **first day of each month**. This regular update schedule ensures subscribers have access to recent voter registration changes.

### One-Year Access Period

The one-year update access period begins when your affidavit is approved and payment is received. During this year:
- Request updates at any time
- Receive monthly refreshed data
- Maintain current voter information
